# Donatelli - Sito (Vite + Tailwind)

Istruzioni rapide:

1. Copia il tuo logo in `public/logo.png` (sostituisci la placeholder). È l'immagine che appare in grande all'apertura.
2. Installa le dipendenze: `npm install`
3. Avvia in sviluppo: `npm run dev`

Note:
- Palette estratta dal logo:
  - Taupe (principale) — #B08A76
  - Deep (accento) — #8D6E5A
  - Cream (sfondo) — #FFFFFF
  - Rose (accento chiaro) — #DCC6B9
  - Charcoal (testo) — #0F1111
- La palette può essere ulteriormente affinata su richiesta.
